✏️ Manipulação de DOM - Aula 2: Modificando Conteúdo

Nesta aula aprenderemos como modificar o conteúdo dos elementos HTML usando JavaScript.

1️⃣ Modificando Texto - textContent

A propriedade textContent modifica apenas o texto do elemento.

Não interpreta HTML, apenas texto puro.

Este é o título original

2️⃣ Modificando HTML - innerHTML

A propriedade innerHTML modifica o conteúdo HTML do elemento.

Interpreta tags HTML e renderiza elementos.

⚠️ Cuidado: innerHTML pode ser um risco de segurança se usado com dados não confiáveis!

Este é um parágrafo original

3️⃣ Adicionando Elementos - appendChild()

O método appendChild() adiciona um novo elemento como filho.

Lista de Compras:

  • Leite
  • Pão

4️⃣ Modificando Atributos - setAttribute()

O método setAttribute() modifica os atributos de um elemento.

Exemplo: src, href, alt, title, etc.



5️⃣ Obtendo Atributos - getAttribute()

O método getAttribute() obtém o valor de um atributo.

📊 Resumo dos Métodos

Propriedade/Método O que faz Exemplo
textContent Modifica apenas texto (sem HTML) elemento.textContent = "Novo texto"
innerHTML Modifica conteúdo HTML completo elemento.innerHTML = "<p>Novo</p>"
appendChild() Adiciona um elemento filho elemento.appendChild(novoElemento)
setAttribute() Define o valor de um atributo elemento.setAttribute('src', 'imagem.jpg')
getAttribute() Obtém o valor de um atributo elemento.getAttribute('src')
