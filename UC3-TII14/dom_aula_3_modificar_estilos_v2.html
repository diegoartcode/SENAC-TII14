🎨 Manipulação de DOM - Aula 3: Modificando Estilos

Nesta aula aprenderemos como modificar os estilos CSS dos elementos usando JavaScript.

1️⃣ Modificando Estilos - Propriedade style

A propriedade style permite modificar estilos CSS diretamente.

Sintaxe: elemento.style.propriedade = "valor"

💡 Nota: Propriedades CSS com hífen (como background-color) viram camelCase em JavaScript (backgroundColor)
Este elemento terá seus estilos modificados

2️⃣ Modificando Múltiplos Estilos

Podemos modificar vários estilos de uma vez.

Elemento com múltiplos estilos

3️⃣ Adicionando e Removendo Classes

Usar classes CSS é mais eficiente que modificar estilos individuais.

Métodos: classList.add(), classList.remove(), classList.toggle()

Elemento para adicionar/remover classes

4️⃣ Verificando Classes - classList.contains()

O método classList.contains() verifica se um elemento tem uma classe.

5️⃣ Criando Animações com Estilos

Podemos criar animações modificando estilos com JavaScript.

📊 Resumo dos Métodos

Método/Propriedade O que faz Exemplo
element.style.propriedade Modifica um estilo CSS elemento.style.backgroundColor = "red"
classList.add() Adiciona uma classe elemento.classList.add('destaque')
classList.remove() Remove uma classe elemento.classList.remove('destaque')
classList.toggle() Alterna uma classe (adiciona se não tem, remove se tem) elemento.classList.toggle('destaque')
classList.contains() Verifica se tem uma classe elemento.classList.contains('destaque')
