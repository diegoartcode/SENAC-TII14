🎯 Manipulação de DOM - Aula 1: Selecionando Elementos + Eventos

Nesta aula aprenderemos como selecionar elementos HTML e responder a ações do usuário usando eventos.

📍 Parte 1: Selecionando Elementos

1️⃣ getElementById() + Evento Click

O método getElementById() seleciona um elemento pelo seu ID.

Depois podemos adicionar um evento onclick para responder quando o usuário clica.

💡 Fluxo: Selecionar → Adicionar Evento → Criar Função

2️⃣ getElementsByClassName() + Evento Click

O método getElementsByClassName() retorna uma coleção de elementos com a classe especificada.

Podemos adicionar eventos a cada elemento da coleção.

3️⃣ querySelector() + addEventListener()

O método querySelector() usa seletores CSS para selecionar o PRIMEIRO elemento.

O método addEventListener() é a forma moderna de adicionar eventos.

💡 Vantagem: Podemos adicionar múltiplos listeners ao mesmo evento!
Passe o mouse sobre mim!

4️⃣ querySelectorAll() + Loop + Eventos

O método querySelectorAll() seleciona TODOS os elementos que correspondem.

Usamos um for ou forEach para adicionar eventos a cada um.
